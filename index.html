Projects


Analysis of shock absorber spring



This was an academic project undertaken in final semester of my engineering degree

  This project dealt with modeling of the helical spring of a shock absorber by using the modeling software Solidworks.Structural analysis was carried out on the spring by using ANSYS software.

Scenario: The analysis is carried out in two methods, first method is by varying the load on the spring and second one is by varying the spring material.

  The analysis is carried out to determine the suitable material for the spring and also to check the maximum load it can sustain. The result obtained by this analysis is compared with analytical results. The analytical result is obtained by calculating the stress and displacement for the spring theoretically.


Exchange rate calculation system



This was a project undertaken by me after learning web developing and web designing

  I had to make a web application consisting of web service that would return the current exchange rate for currencies. It can be used by any bank to compute the value of a money transaction that is given in a foreign currency. Then, respond to the request with total value of the currency.

Scenario: A software company in Los Angeles has many Employees as software engineer from different countries. They are servicing the company on a contract basis. Thus after the termination of the contract as per the agreement, employees might return to their respective countries.

  Their savings need to be converted into their respective country currency, therefore the employee has to request to the bank in which he holds his account to convert his savings to the currency of his respective country. The bank would then use this application to convert the currency, and respond back to employee the value of converted currency.
